# [구현할 내용들]

1. 로그인 회원가입
   - 구글, 카카오, 깃허브 계정 연동해서 회원가입 처리
   - 회원가입 후 처음으로 로그인 했을 경우
     - 닉네임, 직업, 경력, 이메일, 거주지역 정보 받기
2. 회원의 프로필 페이지
   - 모두가 접근가능
   - 회원이 참여한 프로젝트 목록이 출력
   - 회원의 블로그 및 블로그의 게시글 출력
   - 회원의 블로그 게시글은 매일 밤에 서버에서 돌면서 데이터 업데이트
   - 회원의 프로필사진 변경 가능
3. 팀 채팅 기능 (웹 소캣)
4. 구인
   - 스터디, 사이드 프로젝트 구분
   - 구인 내용 입력 (에디터 이용)
   - 오픈채팅방 링크
5. 1:1 문의
   - 문의가 왔을 때 해당 이메일로 메일 발송
